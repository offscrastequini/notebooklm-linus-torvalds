Caderno Temático - Linus Torvalds: O Visionário por Trás do Linux e do Git

> Projeto desenvolvido como parte do desafio da DIO utilizando o Google NotebookLM como ferramenta de apoio aos estudos.

---

 Sobre o Projeto

Este projeto apresenta um caderno temático desenvolvido no NotebookLM sobre a trajetória de **Linus Torvalds**, um dos maiores nomes da história da computação moderna.

O objetivo foi utilizar Inteligência Artificial para organizar informações provenientes de diversas fontes confiáveis, realizar análises, produzir resumos, responder perguntas e criar um guia de estudos sobre sua vida e suas principais contribuições para o mundo da tecnologia.

Além de conhecer sua história, o projeto busca compreender como o Linux e o Git revolucionaram o desenvolvimento de software, a infraestrutura da internet e o movimento Open Source.

---

Objetivos

- Conhecer a história de Linus Torvalds.
- Entender como surgiu o Linux.
- Compreender a criação do Git e sua importância.
- Aprender sobre Software Livre e Código Aberto.
- Utilizar o NotebookLM como ferramenta de aprendizagem.

 Miniguia de Estudos
 Quem é Linus Torvalds?

Linus Benedict Torvalds nasceu em Helsinque, Finlândia, em 28 de dezembro de 1969.

Desde pequeno demonstrava interesse por computadores e programação.

Durante seus estudos universitários iniciou um projeto pessoal que mais tarde se tornaria o Linux.

Atualmente é reconhecido mundialmente como uma das figuras mais importantes da computação.

---

A criação do Linux

Em 1991 Linus decidiu desenvolver um kernel inspirado no sistema Unix.

Inicialmente o projeto era apenas um experimento pessoal.

Após publicar o código na internet, diversos programadores começaram a colaborar.

Esse trabalho colaborativo deu origem ao sistema operacional Linux.

Hoje o Linux está presente em:

- servidores
- smartphones Android
- supercomputadores
- serviços em nuvem
- roteadores
- dispositivos embarcados
- data centers

---

A criação do Git

Anos depois, Linus percebeu a necessidade de um sistema de controle de versões mais eficiente para administrar o desenvolvimento do Kernel Linux.

Assim nasceu o Git.

O Git permite:

- histórico completo das alterações
- trabalho colaborativo
- criação de branches
- fusão de alterações
- rastreamento de mudanças

Hoje é o sistema de versionamento mais utilizado do mundo.

---

 Impacto do Linux

O Linux transformou a computação moderna.

Hoje ele é utilizado por empresas como:

- Google
- Amazon
- Meta
- Microsoft (Azure)
- IBM
- Oracle

Grande parte da infraestrutura da internet funciona sobre servidores Linux.

---
 Impacto do Git

O Git tornou possível o desenvolvimento colaborativo em larga escala.

Praticamente todos os projetos Open Source utilizam Git juntamente com plataformas como GitHub ou GitLab.

 📚 Glossário

 Kernel

Núcleo do sistema operacional responsável pela comunicação entre hardware e software.

---

Linux

Kernel criado por Linus Torvalds em 1991.

---

Git

Sistema distribuído de controle de versões.

---

 Open Source

Modelo de desenvolvimento onde o código-fonte pode ser estudado e modificado.

---

Software Livre

Movimento que defende a liberdade de utilizar, estudar, modificar e distribuir programas.

---

Repositório

Local onde o código é armazenado.

---

 Commit

Registro permanente de alterações realizadas em um projeto.

---

 Branch

Ramificação utilizada para desenvolver funcionalidades sem afetar a versão principal.

---

 Merge

União de alterações entre branches.

---

 GitHub

Plataforma para hospedagem de projetos utilizando Git.
